# Atividade Rust — Estruturas de Dados

## Exercício 1 — Verificar Primeiro
Complexidade: O(1)
Lógica: Retorna o primeiro elemento da lista.
Justificativa: Acesso direto ao índice 0.

## Exercício 2 — Somar Lista
Complexidade: O(n)
Lógica: Soma todos os elementos.
Justificativa: Um loop percorre toda a lista.

## Exercício 3 — Busca Binária
Complexidade: O(log n)
Lógica: Divide a busca pela metade.
Justificativa: Redução logarítmica do espaço.

## Exercício 4 — Pares com Soma
Complexidade: O(n²)
Lógica: Testa todos os pares.
Justificativa: Dois loops aninhados.

## Exercício 5 — Imprimir Pares e Pares
Complexidade: O(n²)
Lógica: Um loop simples + duplo.
Justificativa: O(n) + O(n²) ≈ O(n²)

## Exercício 6 — Potências de Dois
Complexidade: O(log n)
Lógica: Multiplica por 2 até n.
Justificativa: Crescimento exponencial.

## Exercício 7 — Fibonacci Recursivo
Complexidade: O(2^n)
Lógica: Soma recursiva.
Justificativa: Repetição de chamadas.

## Exercício 8 — Bubble Sort
Complexidade: O(n²)
Lógica: Ordena trocando elementos.
Justificativa: Dois loops.

## Exercício 9 — Produto de Matrizes
Complexidade: O(n³)
Lógica: Multiplicação clássica.
Justificativa: Três loops.

## Exercício 10 — Merge Sort
Complexidade: O(n log n)
Lógica: Divide e conquista.
Justificativa: Divisão log + merge linear.
